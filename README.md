# sales-data-analysis
Sales Data Analysis Project

#Objective

This project analyzes a small sales dataset to understand sales patterns across different cities, products, and customers.

# Dataset

The dataset is manually created using Python dictionary and converted into a DataFrame.

Sample Data:

- Date
- City (Pune, Delhi, Mumbai)
- Product (A, B, C)
- Sales
- Customer

# Tools Used

- Python
- Pandas
- Matplotlib

# Analysis Performed

- GroupBy analysis (city-wise and product-wise sales)
- Sales comparison between different cities
- Customer purchase behavior
- Data visualization using graphs

# Key Insights

- Delhi and Mumbai show higher sales compared to Pune
- Product B has consistent demand
- Some customers purchase repeatedly
- Sales patterns vary across cities

# Conclusion

This project demonstrates how basic data analysis techniques can be applied to extract meaningful insights from sales data.
